# lulu

## 项目简介
一个带 CocoaPods 依赖的 iOS 示例工程。

## 目录
- `lulu`
- `lulu.xcodeproj`
- `lulu.xcworkspace`
- `luluTests`
- `luluUITests`
- `Podfile`
- `Podfile.lock`
- `Pods`

## 快速开始
```bash
pod install
```

然后使用 Xcode 打开对应的 `.xcworkspace`。

## 说明
- 这是一个 iOS 工程，常见入口是 `.xcodeproj` 或 `.xcworkspace`。
- 如果项目使用 CocoaPods，先执行 `pod install`。
